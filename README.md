
## 简介

本项目启动于复旦大学的毕业设计，目的是设计一个兼具学习游戏化、多人协同的计算思维在线学习平台。

## 功能介绍
目前已开发的内容如下：
1. 参考blockly-rtc实现的多人协同技术
2. 基于canvas实现的2d游戏场景
3. 基于Google Blockly实现的可视化代码编辑器

# 配置使用环境

## clone项目

git clone https://github.com/Device-astralis/GraduationProject.git

## 安装依赖
cd GraduationProject  
npm install

## 编译运行

npm start

## 打包部署

npm run build
